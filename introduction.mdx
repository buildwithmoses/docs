## What is AirTrigger?

AirTrigger is an AI agent platform you control entirely through Claude Code. Instead of writing code, you describe what you want your agent to do in natural language — AirTrigger handles the infrastructure: queuing, retries, scheduling, and real-time monitoring.

**How it works:**

1. Connect AirTrigger to Claude Code via the MCP server
2. Tell Claude what agent you want to build
3. Claude creates, deploys, and manages it for you
4. Monitor everything from the AirTrigger dashboard

## What can you build?

Use natural language in Claude Code to create agents that:

- Process data pipelines and ETL workflows
- Generate and transform media (images, video, audio)
- Monitor websites and send alerts
- Automate email sequences and marketing workflows
- Scrape and analyze web content
- Run scheduled jobs on cron

All without writing a single line of code.
